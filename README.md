## Requirement

python3 & pip

## Installation

```bash
$ pip install -r requirements.txt
```
To run server with debug mode :
```bash
$ FLASK_APP=app.py FLASK_ENV=development flask run
```

## Run tests
```bash
$ pytest
```

Coverage :
```bash
$ pytest --cov .
```
